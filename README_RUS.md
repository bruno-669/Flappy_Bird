Простая игра имитирующая геймплей всемирно популярной игры.


Используемые библиотеки:
**fmt**- для классического ввода/вывода
**math/rand** - для получения рандомного смещения трубы
**strconv** - для записи чисел типа int в формат строки, для использования в конечном срезе
**strings** - для работ со строками и накопления строки strings.Builder
**time** - для получения задержки в выводе
**keyboard** - для работы ввода с клавиатуры

Управление:
w, space, KeyArrowUp - прыжок вверх
q, ESC - выход
